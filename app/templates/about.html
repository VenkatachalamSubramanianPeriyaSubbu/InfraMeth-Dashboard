About This Project

Project Overview

This project visualizes methane emissions alongside critical infrastructure in the Chicago metropolitan area, using GIS (Geographic Information Systems) technologies. By integrating geospatial data from multiple public sources, the interactive map provides insights into the proximity of emissions to schools, pipelines, power plants, and industrial facilities. The goal is to enable environmental monitoring, infrastructure risk analysis, and informed urban planning.

Data Sources

  • Emissions Data: EDF Methane LeakMap (Environmental Defense Fund) – Methane City Data
  • Facilities (Industrial Sites): EPA TRI Facilities for Chicago
  • Power Plants: ArcGIS Hub – Power Plants in the U.S.
  • Pipelines: U.S. Energy Information Administration – Natural Gas Pipelines
  • School Districts: ArcGIS Hub – School District Boundaries

Technologies Used

  • GIS Tools – GeoPandas, Folium, Shapely for geospatial data analysis and visualization
  • Flask – Lightweight web framework for app deployment
  • Docker – Containerization for scalable cloud deployment
  • AWS Fargate – Serverless compute for hosting the application
  • GIS Concepts – Spatial joins, coordinate projections, interactive mapping
